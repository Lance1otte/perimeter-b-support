Политика конфиденциальности

Настоящая политика конфиденциальности объясняет, как приложение NewsDigest обрабатывает пользовательские данные.

1. Сбор данных

Приложение NewsDigest не запрашивает и не сохраняет персональные данные (ФИО, email, телефон и т.д.). Единственная информация, которая может быть зафиксирована, — это список интересующих пользователя тем, необходимых для персонализации новостной ленты.

2. Использование данных

Темы интересов используются исключительно внутри приложения для формирования персонализированной ленты новостей. Эти данные не передаются третьим лицам, не используются для рекламы или аналитики и не сохраняются в общедоступных источниках.

3. Регистрация

В приложении отсутствует стандартная регистрация. Тематика новостей подбирается вручную по запросу пользователя и закрепляется за созданным для него индивидуальным профилем.

4. Безопасность

Все данные передаются по безопасным протоколам.
